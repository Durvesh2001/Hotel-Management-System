# Hotel-Management-System
Simple console-based CRUD project using core Java collections.

## Structure
- src/Room.java
- src/Guest.java
- src/Reservation.java
- src/HotelManagementSystem.java
- src/Main.java

## How to compile & run
1. Open a terminal in the project root.
2. Compile:
   ```
   javac src/*.java
   ```
3. Run:
   ```
   java -cp src Main
   ```
